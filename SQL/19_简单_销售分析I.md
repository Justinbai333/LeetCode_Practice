SQL Code:

**Solution 1 用窗口函数**

```sql
# Write your MySQL query statement below

SELECT seller_id
FROM Sales
GROUP BY seller_id
HAVING SUM(price) >= ALL(
    SELECT SUM(price) OVER(PARTITION BY seller_id)
    FROM sales
)
```

**Solution 2 不用窗口函数**

```sql
# Write your MySQL query statement below

SELECT seller_id
FROM Sales
GROUP BY seller_id
HAVING SUM(price) >= ALL(
    SELECT SUM(price)
    FROM Sales
    GROUP BY seller_id
)
```

**Note:**

- ALL的用法
- 窗口函数的复习
