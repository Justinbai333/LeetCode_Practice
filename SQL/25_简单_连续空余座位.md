SQL Code:

```sql
# Write your MySQL query statement below

SELECT DISTINCT c1.seat_id
FROM cinema c1 LEFT JOIN cinema c2
ON ABS(c1.seat_id - c2.seat_id) = 1
WHERE c1.free = 1 and c2.free = 1
ORDER BY seat_id
```

**Note**

- 判断连续空余要看前和后，所以要用ABS()
